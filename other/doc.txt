




通信协议：

Output
FF + 键盘包(8B) + FE

Input
55 + AA + type(1B) + len(1B) + data(len/8 B) + sum(1B) + 5A


流式传输
55 + type(1B) + data + AA

数据包传输
55 + type(1B) + data



type：
bit7~4：dest，bit3~0：func
dest：0xF-self，0xE-other，0x1-USB，0x2-BLE
func：1-keybrd，2-mouse，3-point，4-vol，5-dial
